# SetDefaultAppsX v2.1 - Zero-Dependency Achievement

## What Changed

Version 2.1 completes the journey to a truly zero-dependency macOS utility by adding automatic download and installation of **utiluti** with the same security verification we use for swiftDialog.

## New Feature: Automatic utiluti Installation

### Before (v2.0)
Users had to manually install utiluti before running the script:
```bash

# OR download and install the pkg from GitHub

# Then run the script:

```

### After (v2.1)
Just run the script:
```bash

```

## Technical Implementation

### New Function: `install_utiluti()`

```bash
function install_utiluti ()
{
    # Download and install utiluti from GitHub releases
    # Verifies package signature before installation

    logMe "utiluti not installed, downloading and installing"

    expectedUtilitiTeamID="JME5BW3F3R"  # Armin Briegel's Team ID

    # Fetch latest release URL from GitHub API
    LOCATION=$(/usr/bin/curl -s https://api.github.com/repos/scriptingosx/utiluti/releases/latest | \
               grep browser_download_url | grep .pkg | \
               awk '{ print $2 }' | sed 's/,$//' | sed 's/"//g')

    # Download the package
    /usr/bin/curl -L "$LOCATION" -o /tmp/utiluti.pkg

    # Verify Team ID signature
    teamID=$(/usr/sbin/spctl -a -vv -t install "/tmp/utiluti.pkg" 2>&1 | \
             awk '/origin=/ {print $NF }' | tr -d '()')

    # Install only if signature matches
    if [[ "$expectedUtilitiTeamID" = "$teamID" ]]; then
        /usr/sbin/installer -pkg /tmp/utiluti.pkg -target /
    else
        logMe "ERROR: utiluti Team ID verification failed"
        return 1
    fi

    # Cleanup
    /bin/rm /tmp/utiluti.pkg
}
```

### Security Features

1. **Team ID Verification**: Validates that the package is signed by Armin Briegel (JME5BW3F3R)
2. **HTTPS Download**: All downloads use secure connections to GitHub
3. **Automatic Cleanup**: Temporary files are removed after installation
4. **Error Handling**: Clear error messages if download or verification fails

### Updated Function: `check_utiluti_install()`

```bash
function check_utiluti_install ()
{
    logMe "Ensuring that utiluti is installed..."
    if [[ ! -x "${UTI_COMMAND}" ]]; then
        logMe "utiluti is missing - Installing from GitHub"
        install_utiluti
        if [[ $? -ne 0 ]]; then
            logMe "ERROR: Failed to install utiluti"
            logMe "Please install manually from: https://github.com/scriptingosx/utiluti"
            exit 1
        fi
    else
        logMe "utiluti is installed at ${UTI_COMMAND}"
    fi
}
```

## User Experience Improvements

### Truly Zero-Configuration

Version 2.1 achieves the original goal: a script that "just works" out of the box.

**First Run Experience:**
1. User downloads SetDefaultAppsX.sh
2. User runs: `./SetDefaultAppsX.sh`
3. Script automatically:
   - Detects missing swiftDialog → Downloads and installs it
   - Detects missing utiluti → Downloads and installs it
   - Creates necessary directories in `/Users/Shared/SetDefaultAppsX/`
   - Launches the beautiful swiftDialog interface

**No manual steps. No prerequisites. No documentation to read.**

## Error Handling

The script handles several failure scenarios:

1. **Network failure**: Clear error message, suggests manual installation
2. **GitHub API unavailable**: Fallback error message with URL
3. **Team ID mismatch**: Rejects package, provides expected vs actual Team ID
4. **Download incomplete**: Validates file exists before attempting installation

Example error output:
```
ERROR: Failed to get utiluti download URL
ERROR: Failed to install utiluti
Please install manually from: https://github.com/scriptingosx/utiluti
```
